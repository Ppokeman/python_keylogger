<p>1. Download Zip file</p>
<p>2. Extract it</p>
<p>3. Open a Command line</p>
<p>4. <code>pip install pynput</code></p>
<p>5. <code>py Keylogger.py</code></p>
<p>6. Start typing keys</p>
<p>7. A Keylog.txt file will be created on PATH</p>
